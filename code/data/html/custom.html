Create Custom Scripts

Instruction Action Example
KP Performs a keypress or keypress combination on the host machine "KP ENTER" or "KP LEFT_CTRL+LEFT_SHIFT+d"
P Writes the given string on the host machine as if it was a keyboard "P echo hello world"
PLN Writes the given string on the host machine as if it was a keyboard + ENTER at the end "PLN echo hello world"
MM Moves the cursor X Y relative to current position "MM 10,23"
MC Performs a mouse click. Available clicks LEFT and RIGHT "MC LEFT"
EXEC Executes the specified script "EXEC win/deploy_keylogger.ducky"
